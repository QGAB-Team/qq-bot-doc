# 安裝python並安裝機器人必需庫
## 安裝Python
### Windows
在微軟商店下載**python3.10或更高版本**(因爲機器人需要的一些特性只有新版有)
### Linux
Debian
```
apt install python3.10
```
RHEL
```
yum install python3.10  #建議先yum search python3.10一下是否存在www
```
