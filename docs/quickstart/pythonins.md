# 安装python并安装机器人必需库
## 安装Python
### Windows
在微软商店下载**python3.10或更高版本**(因为机器人需要的一些特性只有新版有)
### Linux
Debian
```
apt install python3.10
```
RHEL
```
yum install python3.10  #建议先yum search python3.10一下这个软件包是否存在（doge）
```
没错，要你自己编译，~~但是我懒得给你写教程~~ ~(￣▽￣)~
