# 准备1-下载MCL

# 一键安装版open in new window(建议)

你直接安装iTXTech/mcl-installeropen in new window即可，按照他们的READMEopen in new window来操作

# 无JDK版open in new window

首先，把OpenJDK安装open in new window好

Windows
下载releaseopen in new window，之后运行mcl.cmd即可

Linux(抄的README，但小改了一点)

mkdir mcl && cd mcl
wget https://github.com/iTXTech/mirai-console-loader/releases/download/v2.1.2/mcl-2.1.2.zip    #这里wget你想要的release
unzip mcl-2.1.2.zip     #在mcl文件夹解压下载的release
mcl
chmod +x mcl
./mcl
